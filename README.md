## 🚀 Flagship Project: DAMOSE

Il mio progetto principale è **DAMOSE**, una web app avanzata per il monitoraggio in tempo reale dei mezzi pubblici di Roma (ATAC). Questo progetto unisce tutte le mie competenze:

* **Backend & Big Data:** Gestione e parsing ottimizzato di enormi file GTFS statici e real-time in PHP e MySQL.
* **Mappe Interattive:** Integrazione della libreria Leaflet.js per visualizzare linee e fermate live.
* **UI/UX Moderna:** Interfaccia utente curata, responsive e con un design moderno in stile **Glassmorphism 🧊** (effetto vetro).

⭐️ *Dal parsing di file ZIP pesanti alla creazione di overlay animati, cerco sempre di trasformare problemi complessi in interfacce semplici e pulite.*
